## About
This is the third edition of PowerBrowser, also nicknamed as PowerBrowser Christmas Edition. It uses the same WebEngine as Version 2 with the same framework, but the main overhaul is designing of the browser. This edition includes a much more refined, processed and complete version of the application with each aspect of the browser including animations and visual elegance. It also includes a new homepage continuing on the idea of the 1st version of the browser.


## Upgrades
- Overhaul of UI
- Adding of History and Bookmarks Features in their own separate dialog(s)
- Advancement of Status Bar
- Including of tricks on URL bar

## Expectations in v4
- Added a error log for any occurring of errors
- Separation of Settings from Menu Bar to a new panel
- Overall adapts a more simplistic UI
