# PowerBrowser 5
A Rich PyQt6 Web Browser

## Information
PowerBrowser is a self-made Python based Web Browser made by Toshan Chowdhury. It originally was made for a 9th grade personal project but it is available for public usage if needed. This version marks a major milestone as the 5th version has been released of this browser.

## Upgrades
- A moernized look based from the previous v4.
- A new system of storing data like History and Bookmarks in SQLite.
- Introduces a new "About" Tab
