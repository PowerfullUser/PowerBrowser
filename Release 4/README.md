# PowerBrowser v4

### About
The fourth edition of PowerBrowser improves from the third by simplying and decluttering the entire UI from version three and making it look cleaner. Overall, there are no new changes other than the new mute site button allowing the user to mute and unmute the site at any given moment.

### Requirements
Required Libraries:
PyQt6
PyQt6-WebEngine

Insert this into command prompt for downloading the source code:
``pip install PyQt6 PyQt6-WebEngine``
