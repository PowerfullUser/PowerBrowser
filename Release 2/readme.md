# PowerBrowser 2
The new upgraded version better than ever before

## What is it?
PowerBrowser 2 is a completely overhaulled version from PowerBrowser v1. It is more optimized thanks to the upgrading of PyQt5 to PyQt6, below will be all improvements from the previous version.

## List of Improvements
- Added download progress bar for the downloading of any item.
- Including of history features onto the browser, tracking browser history to the nearest millisecond.
- Provides a usefull bookmark feature for adding your favourite or required websites.

## How to Donwload?
To download this version of PowerBrowser, you will need to download the following librarires listed in the set provided below:

```
pip install PyQt6 PyQt6-WebEngine
```
