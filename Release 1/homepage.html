PowerBrowser


Information of Browser

Welcome to PowerBrowser, it is a Simple Web Browser designed to fullfill the needs of basic users, examples are watching youtube, playing simple games, programming on replit, making documents in Google Docs / Microsoft Word. If you are willing to perform high-end tasks on this web browser, please wait until November for a more updated version of Chromium in this browser to load higher-end tasks. Hope you enjoy!


About Person

Toshan Chowdhury, born on January 3, 2012 is the creator of this following Web Browser. At the age of 13, this project was started in the month of September. The first official release was in October 2025. The original purpose of this project was to be a personal project for his school in year 9 in Finland.


About Project

The Project has been created using PyQt5. The Web Engine running the websites is a slightly outdated version of Chromium 87.0.4280.144. While this browser will not be able to load high-end websites, it is certain to capable of rendering low and medium end websites. The main purpose of this browser however is to be able to surf the internet freely while being quick without any hastle on any end
